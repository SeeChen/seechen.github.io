如何打开文件

您可以使用快捷键 Crtl + O 来打开媒体文件。

或者您可以使用右键以及菜单栏中的 Open file 栏目以打开媒体文件:

其实，在没有媒体文件播放的时候您可以直接点击 播放 按钮来打开您要打开的媒体文件

还有还有，你也可以直接拖动媒体文件到应用里面来播放吖。


菜单栏以及右键菜单

菜单一共被划分为三个区域，即 File , View , Help , 以及 Settings 。

在 File 的子菜单中，有一个 Play List 的选项，当你打开视频并点击此菜单后，将会显示一个播放列表。

在你把多个媒体文件打开后，你就可以直接从这里跳转啦。

当然，除了使用菜单的选项，你也可以直接点击这里:

或使用 Crtl + P 来打开播放列表。



点击 File 的第三个子菜单后，将会显示您曾经播放的历史记录：

而每个记录分别有：

01.    您最后播放这个视频的时间

02.    播放的内容路径

03.    最后播放停止的位置

04.    媒体文件的类型

05.    播放按钮以及删除按钮： 播放按钮可以让你从上次观看到的地方继续播放，但如果上次是已经播放结束了，则会重新播放；而删除按钮则可以删除掉之一行的历史记录。



最后是两个顶部的按钮： Delete All 和 Close

Delete All ： 将所有的历史记录删除。


Close ： 关闭历史记录页面，不会删除历史记录。
